astrolibrary.data_visualization.spectral_visualization module¶

astrolibrary.data_visualization.spectral_visualization.plot(data, window_size=1)¶
Parameters:

  • data (pd.DataFrame) – Dataframe containing spectral data from the SDSS catalog

  • attributes (list) – List of attributes that will be extracted from the ‘data’ dictionary

Return type:

A matplotlib that displays the spectral data with inferred continuum

Examples

>>> data = pd.DataFrame(
                {
                    "wavelength": np.array([1,2,3,4]),
                    "flux": np.array([1,2,3,4]),
                    # More stuff
                }
>>> vs = Visualizer(type="Inferred Continuum")

# This should plot/show the graph, without saving x >>> vs.plot_spectra(data, window_size = 0.5, should_plot=True)

# This should save plot and return filename, without plotting it >>> vs.plot_spectra(data, type=”Inferred Continuum”, should_plot=False) )
